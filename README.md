# Smart Industrial Maintenance Platform

## Description

Smart Industrial Maintenance Platform is a web-based intelligent maintenance system that uses Artificial Intelligence to monitor industrial equipment, analyze operational data, detect anomalies, and predict potential failures.

## Technologies

* React.js (Frontend)
* Spring Boot (Backend)
* MySQL (Database)
* Flask & Python (AI Service)

## Features

* Equipment Management
* Maintenance Tracking
* Alert Management
* Anomaly Detection
* Failure Prediction
* AI Recommendations
* Dashboard & Reports

## Project Structure

```text
frontend_fixed/   -> React Frontend
backend/          -> Spring Boot Backend
ai-service/       -> Flask AI Service
```
